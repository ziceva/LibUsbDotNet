Interface IUsbContext

Initializes a new libusb context. You can access most of libusb's functionality through context. Multiple contexts operate in isolation from each other.

List()

Returns a list of USB devices currently attached to the system.

Declaration
UsbDeviceCollection List()
Returns
Type Description
UsbDeviceCollection

A UsbDeviceCollection which contains the devices currently attached to the system.
Remarks

This is your entry point into finding a USB device to operate.

You are expected to dispose all the devices once you are done with them. Disposing the UsbDeviceCollection will dispose all devices in that collection. You can Clone() a device to get a copy of the device which you can use after you've disposed the UsbDeviceCollection.
