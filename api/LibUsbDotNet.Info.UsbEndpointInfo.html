Attributes

Attributes which apply to the endpoint when it is configured using the bConfigurationValue.

Declaration
public virtual byte Attributes { get; }
Property Value
Type Description
System.Byte
Remarks
  • Bits 0:1 determine the transfer type and correspond to EndpointType.
  • Bits 2:3 are only used for isochronous endpoints and correspond to IsoSyncType.
  • Bits 4:5 are also only used for isochronous endpoints and correspond to IsoUsageType.
  • Bits 6:7 are reserved.
